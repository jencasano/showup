Mixtape · mobile exploration · M1 / M2 / M3

showup. / three mobile variants, Side A

Based on the actual mobile layout in the live app: 7-col month-grid calendar card, amber-dot diary indicator, diary widget below, bottom nav with 3 tabs. Each variant below shows the same My log view rendered with different levels of Mixtape personality. Both 390px (iPhone 14 size) and 360px (smaller Android) widths shown side by side.

M1
Quiet mobile
thin stripe · clean header · minimal flourish
The pitch. Just enough Mixtape to be recognisable: thin 3px color stripe under the header, nothing else. No track numbers, no motif, no ink-pill on the bottom nav. The redesign is carried entirely by new tokens — palette, typography, texture. Mobile fades into the background and lets the content breathe. Lowest risk and best performance on cheaper Android devices.

M2
Loud mobile
stripe · motif + track number in header · ink-pill bottom nav
The pitch. Maximum Mixtape personality. Small motif dots above the logo echoing the desktop sidebar motif. A tiny mono track number ("01 / my log") tells you where you are. Bottom nav active state is a full ink pill with the active track number + red accent — same visual language as the desktop sidebar active pill. Most brand-dense but also the most visually loud on small screens.

M3
Balanced mobile
stripe · clean header · ink-pill active state on bottom
The pitch. Takes the ink-pill active state from M2 (so the bottom nav has Mixtape personality), but drops the track numbers and the motif dots in the header. Header stays quiet, content area is where the motifs drift in the background. Compromise between restraint and brand. Recommended pick if you want Mixtape to feel present but the content to stay the star.

NOTES
Reading the three
one of these becomes the mobile Mixtape
M1 Quiet. Conservative. Looks like any well-made mobile app. Easy to ship, easy to maintain, least performance cost, works fine on old Android. Downside: wouldn't scream "Mixtape" if you saw it without the logo.

M2 Loud. Strongest brand presence. The track numbers in the bottom nav + the motif dots + the ink pill all tell you "this is showup., not another tracker." Risk: can feel busy on 360px width, and the tiny "01/02/03" labels get cramped next to "Following" which is longer than the others.

M3 Balanced. My pick. Bottom nav has the ink-pill personality (so it feels clearly Mixtape, not generic), header stays quiet to respect the small screen. No fiddly track-number text in places that don't need it. This is the closest mobile equivalent to the locked desktop V3 sidebar: personality concentrated where nav happens, content area let breathe.

None of these show Side B yet. Once you pick, I'll add a Side B render of the winner so we see both faces.
