The tradeoffs:
  • Red. Conflicts with error which also uses red. Success and error would be the same color, distinguishable only by copy. Bad signal hierarchy. Unless we downgrade error to something else, red is out for success.
  • Green. On Side A, --green #3E5C3A is a deep forest that reads confident without shouting. On Side B, --green auto-remaps to #5EAAA8 (teal) which is the warmer companion. Works on both. Traditional but not dated.
  • Yellow. Same hex (#E8B33A) on both themes, which is unusual but the yellow was chosen to work on both paper colors. Reads as "nice" rather than "pass/fail." Risk: conflicts with the future warning type (which is also yellow).
