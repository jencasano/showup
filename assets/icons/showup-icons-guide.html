03 — Setup

How to Add to showup.

1
Pull the latest code — the icons are already in the repo at /assets/icons/.
2
Link the CSS in index.html:
<link rel="stylesheet" href="css/icons.css" />
3
Import the JS helper in any JS file that needs icons:
import { icon, iconEl, iconSrc, STICKER_ICONS } from './icons.js';
4
Link the favicon in <head> — already done in index.html:
<link rel="icon" type="image/svg+xml" href="/assets/icons/favicon.svg" />
04 — Usage

Where to Use Them

Anywhere you'd use an emoji, use an icon instead.

Month Bar — Streak Stat

3 day streak · 3/4 days done 
// In app.js updateStat() — use .innerHTML not .textContent

monthBarStat.innerHTML = parts.join(' · ');

05 — Notes

A Few Things to Know

Sticker value in Firestore
Store the icon name string: "sunflower", "flame", etc. Existing users will need to re-pick their sticker on next monthly setup.
innerHTML vs textContent
When inserting icons via icon(), always use .innerHTML — never .textContent.
Adding new icons
Drop a new .svg into /assets/icons/, add its name to ICON_NAMES in js/icons.js. Done.
Firebase hosting
/assets/ deploys automatically with firebase deploy. No extra config needed.
